# Kleiderspenden-Registrierung
Eine responsive Webanwendung (Single-Page-Application) zur Erfassung von Kleiderspenden.
Entwickelt für die Fallstudie des fiktiven Kleiderspenden e.V.

## Umgesetzte Funktionen
* **Globale Navigation & Branding:** Eine übersichtliche Kopfzeile mit lokal eingebundenem Logo, das als funktionaler Home-Button dient, sowie Platzhalter-Links für weitere zukünftige Hauptbereiche.
* **Formular:** Erfassung von Spendenart, Krisengebiet und Übergabeart (Geschäftstelle oder Abholung).
* **Validierung:** Das Formular prüft bei der Eingabe in Echtzeit, ob die Postleitzahl für das Sammelfahrzeug aus genau 5 Ziffern besteht und mit "12" beginnt.
* **Barrierefreiheit:** Screenreader-Unterstützung bei der Bestätigungsansicht für eine bessere Zugänglichkeit.
* **Responsive Design:** Die Benutzeroberfläche passt sich mithilfe von CSS-Flexbox und Bootstrap automatisch an alle Bildschirmgrößen an. Dies umfasst ein aufklappbares Hamburger-Menü für Smartphones sowie einen "Sticky Footer" für ein sauberes Layout bei kurzen Inhalten.

## Verwendete Technologien
* HTML5 / CSS3
* Vue.js 3 (via CDN eingebunden)
* Bootstrap 5 (via CDN eingebunden)

## Ausführung des Projekts
Um die Anwendung zu starten, laden Sie die Dateien dieses Repositories (`index.html` und `logo.png`) herunter und **speichern Sie zwingend beide Dateien im selben Ordner**.
Öffnen Sie anschließend die `index.html` in einem aktuellen Webbrowser (z. B. Chrome, Firefox oder Edge). Es ist keine weitere Installation notwendig.
